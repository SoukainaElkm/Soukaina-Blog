EL KAMOUNI Soukaina's Blog



Dans mon Projet Blog - Express, j'ai créé une application Web basée sur le framework Express. Nous avons aussi accées à une base de données MySql, où sera stocker toutes les information concernant les utilisateurs, les articles, les tags, et les commentaires. Ces derniers seront générés par Faker, et l'utilisateur aura la possibilité d'ajouter, modifier et supprimer toutes les informations.



Les entités utilisées dans l’application sont :

  • User, (id, username, email, password, role)
  • Article (id, title, content, published)
  • Comment (id, content)
  • Tag (id, name)
    • Le champ role peut avoir l’une de ces trois valeurs (admin, author ou guest)

    Les règles et associations :

  • Un utilisateur (User) peut créer plusieurs articles (Un article est associé à un seul utilisateur)
  • Un article (Article) peut avoir plusieurs Tags
  • Une Tag (Tag) peut apparaître dans plusieurs articles
  • Un article peut avoir plusieurs commentaires (Comment) (Un commentaire est associé à un article)


    • Package utilisés :

  • Express : Framework web (le projet sera généré à l’aide la CLI express-generator)
  • Middlewares Express : selon le besoin
  • Sequelize : ORM (Object Relational Mapping) avec le driver MySQL (mysql2)
  • Visual Studio Code avec l’extension Rest Client pour tester les routes
